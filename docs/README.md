# lecouRAGE DN42

## Overview

This repository contains the design, implementation and operation of a DN42 Autonomous System using:

- FRRouting (FRR)
- WireGuard
- eBGP
- iBGP
- MP-BGP
- IPv4
- IPv6
- EVPN
- MPLS
- Service Provider Architectures
